# Mela Monitor + Mela Atlas Execution Plan

Saved from physician review panel feedback. See full plan below.
Status: APPROVED for execution.

This document is the master execution plan. ADR-130 (Dr. Chang feedback)
is incorporated. This plan supersedes all prior priority lists.

## Key Strategic Decision

Split into THREE streams:
1. Mela Monitor — non-diagnostic monitoring app (ship first)
2. Mela Atlas — molecular melanoma research platform
3. Mela Validation Program — evidence and partner infrastructure

## Execution Order (non-negotiable)

1. Lock claims and evidence
2. Lock datasets and leakage policy
3. Build capture-quality and abstention
4. Ship Monitor v1
5. Build Atlas v1
6. Start external pilots
7. Revisit regulated device strategy only after real evidence justifies it
